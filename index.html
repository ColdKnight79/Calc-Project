Calculations

Creating an equation

The first thing needed to do any work is to find the equation the represents the situation. in this project i will be representing the road as 2 triangles. Since im working in an 8x8 area i can say that the side x of one of the triangles plus the side y of the other is equal to 8. I also know that the vertical side of the top triangle is the length of the marshland and the vertical side of the bottom triangle is the size of the land. from there you rearange the x + y = 8 base of the triangle to be y = 8-x. that is the new base of the second triangle and you end up with the triangles bellow.

Finding The Final Equation

Now that i have the sides of the triangle using pythagorean theorem it is possible to create an equasion for the cost of the road. this is done by rearanging \( a^2 + b^2 = c^2 \) into \( c = \sqrt{a^2 + b^2} \). from there i can sub in the triangles and get y = \(\sqrt{5^2 + X^2} + \sqrt{3^2 + (8 - X)^2}\) finaly i need to multiply the equations for the hypotenuse of each triangle by their respective cost to build. That creates the final equation of y = \(10 * \sqrt{5^2 + X^2} + 7 * \sqrt{3^2 + (8 - X)^2}\)

Finding The Minimum Cost

From this point the only way to find the minimum value is to find the 0 of the first dirivative the first step in deriving this equation is using the chain rule on the \(10 * \sqrt{5^2 + X^2} \) so that I can power rule the square root and \(X^2\) to create \( \frac{10X}{\sqrt{25 + X^2}} \) I then do the same thing with the \(7 * \sqrt{3^2 + (8 - X)^2}\) to create \( \frac{-7(8 - X)}{\sqrt{9 + (8 - X)^2}} \), these are then put back togethe to make \( y' = \frac{10X}{\sqrt{25 + X^2}} + \frac{-7(8 - X)}{\sqrt{9 + (8 - X)^2}} \). That dirivative is not able to be simplifed so the only way to find the minimum is to graphicaly find the 0 of y'. The graph of the dirivative shown on the right shows that the minimum cost will occur at X = 3.56005. From this point the math to find cost and lenght of road is found in the graphs section.

Conclusion

In Conclusion The 3 possible Scenarios provide their own angles both literaly and figuratively. The cheapest Solution is extremely enticing being 0.19 miles longer than the shortest possible solution while being 1.5 million dollars cheaper. That said the shortest solution also comes with the benefit of no turn in the shorter road. That lack of turn coupled with the shorter road might help to speed up trafic, reduce maintence cost and speed up construction. Option 3 is a bit of a wild card that provides an interesting situation where the road is run straight down to exit the marsh in as short as possible time. Marsh land is often soft and unstable and building a long road on it might open up excessive maintenance costs. For the extra 28 million to build option 3 it might be worth it if the saving in maintenence cost if this is a high trafic highway that will be worn out very quickly. With the currently known information I would sujest the city goes with the first Scenario since it is the cheapest and the extra length is insignificant to the time spend drivng down the road at highway speed.
